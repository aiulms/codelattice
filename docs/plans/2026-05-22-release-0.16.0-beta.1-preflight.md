Date: 2026-05-22
Base: `afe043f` (`fix(mcp): reduce default AI toolset to six entries`)

## Goal

Refresh the public beta release so the downloadable artifact matches current CodeLattice capabilities: JavaScript, Shell, workspace intelligence, WebUI workbench, native detect-changes, and the AI-friendly six-tool MCP default surface.

## Release Version

Use product version `0.16.0-beta.1`.

Rationale: `v0.15.0-beta.1` is already tagged and published from an older commit. Current master adds multiple compatible user-facing capabilities after that tag, including new language support and MCP/WebUI workflow changes. Per `docs/release-versioning.md`, this is a beta minor release.
