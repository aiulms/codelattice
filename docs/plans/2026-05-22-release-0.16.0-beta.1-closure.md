# CodeLattice 0.16.0-beta.1 Release Refresh Closure

Date: 2026-05-22
Base: `afe043f` (`fix(mcp): reduce default AI toolset to six entries`)

## Summary

Prepared CodeLattice `0.16.0-beta.1` as the next beta release candidate after `v0.15.0-beta.1`.

This release refresh aligns the packaged artifact, installer docs, release notes, smoke matrix, and MCP/WebUI scripts with the current product surface:

- 9 analyzed language paths in the packaged smoke matrix: Rust, Cangjie, ArkTS, TypeScript, JavaScript, C, C++, Python, Shell.
- AI-friendly default MCP toolset remains 6 facade tools.
- Full MCP toolset remains available via `CODELATTICE_MCP_TOOLSET=full` and currently exposes 49 tools.
- WebUI runner snapshot enrichment can internally use full MCP mode for automation graph enrichment without widening the default AI toolset.

## Fixes Made During Release Gate

- Added Shell fixture packaging and Shell release-smoke coverage.
- Added JavaScript/Shell profile fields to release manifests and release docs.
- Updated release installer defaults from older beta versions to `v0.16.0-beta.1`.
- Synced dogfood/install/live contract thresholds with the current 49-tool full profile and 6-tool default AI profile.
- Fixed WebUI runner automation-graph enrichment so it explicitly uses full toolset internally while leaving public `/api/mcp/tools` at the 6-tool facade default.
- Added current-version release notes to release metadata checks and package staging.

## Risk Review

This is expected for this release gate because the changed files include cross-cutting release scripts, installer docs, package manifest logic, and WebUI runner behavior. The risk is not ignored: all affected release/MCP/WebUI paths were covered by targeted smoke tests above.
